# TITLE : 2a.) Display the Student
```
lass student1{
int id;
String name;
void setdata(int i,String n){
id =i;
name=n;
}
void display(){
System.out.println("student1 ID:"+id);
System.out.println("student1 name:"+name);
}
public static void main(String[] args){
student1 s1=new student1();
s1.setdata(101,"sai");
s1.display();
}
}
```
